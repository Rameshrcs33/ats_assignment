# Expo OTP Authentication App

A React Native application built with **Expo** featuring a secure **OTP (One-Time Password) verification screen** with a **60-second resend OTP timer**.

---

## Features

- OTP input screen
- Resend OTP enabled after **60 seconds**
- Mobile-friendly UI
- Built with Expo for fast development and testing

---

## Tech Stack

- **React Native**
- **Expo**
- **TypeScript**
- **React Hooks**
- **Expo CLI**
